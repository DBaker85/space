# Personal Website

My personal fun website that is a super overkill MERN stack

---

## Based on

- NodeJS hosted on HEROKU
- MongoDB hosted on MONGOLAB
- ExpressJS
- GraphQL
- ReactJS
- GSAP

---

## Built with:

- Typescript
- Prettier.io
- Travis-CI
- Create-react-app
